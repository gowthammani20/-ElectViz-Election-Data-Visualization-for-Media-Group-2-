# -ElectViz-Election-Data-Visualization-for-Media-Group-2-

ElectViz – Election Data Visualization for Media

ElectViz is a data analytics and visualization project developed during my internship to analyze and present election data in an interactive and meaningful way for media analysis and reporting. The project focuses on transforming raw election datasets into structured analytical models and dashboards that help understand electoral trends across states.

Project Objective

The main goal of this project is to integrate multiple election datasets, design a structured data model, and build dashboards that provide clear insights into electoral patterns. The project demonstrates skills in data integration, data modeling, dashboard creation, and analytical storytelling.

Project Milestones

Milestone 1 – Data Collection & Preparation:

Collected and organized election datasets for:

Lok Sabha Elections – Tamil Nadu & Odisha

Lok Sabha By-Elections – Tamil Nadu & Odisha

Vidhan Sabha Elections – Tamil Nadu & Odisha

Vidhan Sabha By-Elections – Tamil Nadu & Odisha

Cleaned and structured datasets for further analysis.

Milestone 2 – Dashboard Development:

Created an Email Marketing Customer Segmentation Dashboard as the first analytical presentation to demonstrate dashboard development and data visualization capabilities.

Built initial dashboards to represent election insights.

Milestone 3 – Data Integration:

Integrated State Level By-Poll datasets as part of the group task.

Combined multiple datasets to create a unified analytical dataset.

Milestone 4 – Data Modeling:

Designed a Star Schema Data Model for efficient analysis.

Developed the first dashboard in the team project the integrated dataset and data model.

Final Internship Deliverables

Conducted two project presentations explaining the analytical workflow and dashboards.

Performed two live demos demonstrating the final integrated dashboard and data model.

Collaborated with the internship team to deliver the final analytics solution.
